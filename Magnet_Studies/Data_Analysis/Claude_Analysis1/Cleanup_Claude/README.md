# LDRD FFA@CEBAF Magnet Measurement Data — Cleaned & Merged

## Overview

This directory contains deduplicated, merged magnet measurement data from the
LDRD permanent magnet radiation study at CEBAF. Data spans from pre-deployment
baselines through post-run measurements (2024-04-24-09:52:48 to 2026-01-12-12:22:40).

Generated by `merge_script.py` — see that script for full provenance details.

## Data Format

All files are tab-separated `.dat` files with two possible row formats:

**Helmholtz (old format, early pre-deployment):**
```
 YYYY-MM-DD-HH:MM:SS\t\t\t<value> kT
```
Note: 'kT' units appear in only ~7 very early rows (May 2024). These may
reflect an early DAQ configuration before standardization.

**Helmholtz (standard format, most data):**
```
 YYYY-MM-DD\tHH:MM:SS\t<dosimeter_serial>\t<rod_id>\t<value> mWC
```

**Teslameter (readings in mT):**
```
 YYYY-MM-DD-HH:MM:SS\t<serial>\t<rod>\t<field1>\t<field2>\t<field3>\t<temp>
```
or with separate date/time tabs in newer data.
Fields 1-3 are magnetic field components in mT; field 4 is temperature in °C.

## Sentinel Values

Rows with `1337` readings have been **excluded**. These indicate no measurement
was taken (dosimetry swap or testing in progress), NOT instrument failure.

## Deduplication Method

1. All versions of each sample file collected from all sources
2. Each row parsed for datetime key (normalized across old/new date formats)
3. Rows with `1337` sentinel values excluded
4. Rows with empty/missing readings excluded
5. Union of all unique (datetime) rows, sorted chronologically
6. On timestamp conflict (different readings), longer/more-complete row kept
