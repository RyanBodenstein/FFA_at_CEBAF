# 07_Reference_Documents

Material specifications and instrument documentation.

## Files

### allstar_magnetics_specs.md
Verified specifications for the four magnet grades used in this study,
sourced from Allstar Magnetics datasheets. Includes magnetic properties
(Br, Hc, Hci, BHmax), temperature coefficients, maximum operating
temperatures, and radiation resistance predictions (delta T_crit).

### instrument_specs.md
Specifications for the two measurement instruments:
- **Helmholtz coil fluxmeter** (Fluxmeter Model 2130): DC accuracy,
  measurement ranges, drift characteristics, and session-to-session
  gain variability measured in this study.
- **Teslameter** (SENIS 3MH6-E with Type C Hall probe): accuracy,
  temperature stability, probe specifications, and positioning
  systematic uncertainties measured in this study.
