PC BASED OPTI-CHROMIC READER SYSTEM
MODEL FWT-200S
 

The Model FWT-200S Opti-Chromic Reader System offers a state of the art performance for both high and low volume users of Opti-Chromic Dosimeters. This reader is fast, accurate and above all easy to use.

The system consists of a reader head, PCI interface card that plugs into the inside of your PC and assorted programs. 

The addition of the computer to the measurement cycle reduces the complexity of the system. The computer does all the calculations necessary not only to take a reading but also to reduce the data and store it for later reference or reporting. It assists the user by performing the following tasks:

Automatic Zero
The computer will set the zero at the user’s request.
Data reduction: The computer will accept the user calibration data and report the calculated dose using all necessary parameters.

Software
Beginning with version 2.0 the software is compatible with Win 95/98/ME/2000/XP. The windows version of the software is a dynamic data exchange (DDE) server. It provides information on the absorbance readings for both the 605 nm and the 656 nm analysis wavelengths to DDE client programs such as Microsoft Excel or Access. A sample Excel spreadsheet is included to illustrate how data may be accessed by Excel's DDE functions. Due to the critical timing needed to measure absorbance it is best if the DDE server and client are the only programs running on the computer.

Operational
The Opti-Chromic head has a holder located under a cover. Place the dosimeter in the holder and close the lid. The computer will signal when the reading is complete.
